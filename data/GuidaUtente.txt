GUIDA UTENTE - PROGRAMMA GESTIONE BAR

MENU PRINCIPALE
1. Aggiungi ordine
2. Estrai ordine  
3. Visualizza ordine
4. Leggi tutti gli ordini
5. Mostra ordini di un tavolo
6. Salva su file
7. Carica da file
8. Mostra guida utente
0. Esci

COME USARE OGNI OPZIONE

1. Aggiungi ordine
   - Inserisci N. Tavola + Bevande + Quantita
   - Es: Tavola 3 - Cappuccino x2 - Birra x1
   - Ordine va IN FONDO alla coda

2. Estrai ordine
   - PRELEVA e ELIMINA il PRIMO ordine
   - Stampa dettagli completa
   - Se vuota: "Nessun Ordine Presente"

3. Visualizza ordine
   - MOSTRA PRIMO ordine SENZA toglierlo
   - Controllo rapido testa coda

4. Leggi tutti gli ordini
   - Elenca TUTTA la coda
   - [Tavola N] - Bevande + Quantita
   - Premi tasto per paginazione

5. Mostra ordini di un tavolo
   - Inserisci N. Tavola (es: "3")
   - Mostra SOLO ordini di quella tavola

6. Salva su file
   - Salva TUTTA coda su ordini.dat
   - Backup completo FIFO preservato

7. Carica da file
   - Svuota coda + ricarica da ordini.dat
   - Ripristina lavoro precedente

8. Mostra guida utente
   - Visualizza questa guida

0. Esci
   - Termina programma

NOTE IMPORTANTI
- Schermo si pulisce automaticamente
- Premi un tasto dopo ogni operazione  
- File dati: ordini.dat (stessa cartella)
- Coda FIFO: Primo arrivato = primo servito

TEST RAPIDO
1 - Aggiungi 2 ordini - 4 - Vedi tutti
6 - Salva - 0 - Esci - Riavvia
7 - Carica - 4 - Verifica identico!
